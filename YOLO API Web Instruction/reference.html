ImageNet Classification

Darknet을 사용하여 1000-Class ImageNet 챌린지에 대한 이미지를 분류 할 수 있습니다ImageNet challenge. 아직 다크 넷을 설치하지 않았다면 먼저 다크 넷을 설치해야합니다..

Classifying With Pre-Trained Models

다음은 Darknet을 설치하고 분류 가중치 파일을 다운로드하고 이미지에 분류자를 실행하는 명령입니다.

git clone https://github.com/pjreddie/darknet.git
              cd darknet
              make
              wget https://pjreddie.com/media/files/extraction.weights
              ./darknet classifier predict cfg/imagenet1k.data cfg/extraction.cfg extraction.weights data/dog.jpg

이 예제에서는 추출 모델을 사용하여 자세한 내용을 볼 수 있습니다.예제. 이 명령을 실행하면 다음과 같은 결과가 나타납니다.:

Darknet은 구성 파일과 가중치를로드 할 때 정보를 표시 한 다음 이미지를 분류하고 이미지의 상위 10 개 클래스를 인쇄합니다. 우리는 이것을 성공으로 생각합니다!!

대머리 독수리 이미지와 같은 다른 이미지로 시도 할 수도 있습니다.:

이미지 파일을 지정하지 않으면 런타임에 이미지를 묻는 메시지가 나타납니다. 이렇게하면 전체 모델을 다시로드하지 않고 여러 행을 연속으로 분류 할 수 있습니다.. Use the command:

./darknet classifier predict cfg/imagenet1k.data cfg/extraction.cfg extraction.weights

그러면 다음과 같은 프롬프트가 나타납니다:

....
              27: Softmax Layer: 1000 inputs
              28: Cost Layer: 1000 inputs
              Loading weights from extraction.weights...Done!
              Enter Image Path:

이미지 분류에 지루할 때마다 Ctrl-C Ctrl-C 를 사용하여 프로그램을 종료 할 수 있습니다.

Validating On ImageNet

어디에서나 이러한 유효성 검증 세트 번호가 표시됩니다. 어쩌면이 모델들이 실제로 얼마나 잘 작동하는지 스스로 확인하고 싶을 것입니다. 그것을 해봅시다 !

먼저 유효성 검사 이미지와 cls-loc 주석을 다운로드해야합니다. 주석은 여기에서다운가능합니다 여기에서 얻을 수는 있지만 계정을 만들어야합니다! 모든 것을 다운로드하고 나면 ILSVRC2012_bbox_val_v3.tgz, 및 ILSVRC2012_img_val.tar. 가있는 디렉토리가 있어야합니다. 먼저 압축을 풉니 다.:

tar -xzf ILSVRC2012_bbox_val_v3.tgz
              mkdir -p imgs && tar xf ILSVRC2012_img_val.tar -C imgs

이제 이미지와 주석이 있지만 Darknet이 예측을 평가할 수 있도록 이미지에 라벨을 지정해야합니다.. 이 bash 스크립트 하위 디렉토리를 사용하면됩니다.. 이미 스크립트 / 하위 디렉토리/ 에 있습니다. 우리는 다시 그것을 얻을 수 있고 그것을 실행할 수 있습니다 :

wget https://pjreddie.com/media/files/imagenet_label.sh
              bash imagenet_label.sh

이것은 두 가지를 생성 할 것입니다 labelled/ 이미지로의 이름이 변경된 심볼릭 링크를 포함하는 디렉토리와 라벨이 붙은 이미지의 경로 목록을 포함하는 inet.val.list 파일. 이 파일을 inet.val.list 서브 디렉토리로 이동해야합니다. data/ 

mv inet.val.list <path-to>/darknet/data

이제 마침내 모델을 검증 할 준비가되었습니다! 먼저Darknet을 다시 만듭니다 . 그런 다음 유효성 검사 루틴을 다음과 같이 실행하십시오.

./darknet classifier valid cfg/imagenet1k.data cfg/extraction.cfg extraction.weights

Note: OpenCV로 Darknet을 컴파일하지 않으면 ImageNet 이미지 중 일부가 stb_image.h에서 지원되지 않는 별난 형식이기 때문에 이미지 넷 이미지를 모두로드 할 수 없습니다..

CUDA로 컴파일하지 않으면 ImageNet에서 유효성을 검증 할 수는 있지만 오랜 시간이 걸릴 것입니다. 권장하지 않습니다.

Darknet Reference Model

이 모델은 작지만 강력하도록 설계되었습니다. 그것은 AlexNet과 동일한 top-1 및 top-5 성능을 달성하지만 매개 변수의 1/10은 달성합니다. 끝에 대부분 큰 연결 레이어가없는 대부분 길쌈 레이어를 사용합니다. CPU에서 AlexNet보다 약 2 배 빠르기 때문에 일부 비전 응용 프로그램에 더 적합합니다.

  • Top-1 Accuracy: 61.1%
  • Top-5 Accuracy: 83.0%
  • Forward Timing: 1.5 ms/img
  • CPU Forward Timing: 0.16 s/img
  • cfg file
  • weight file (28 MB)

VGG-16

TOxford의 Visual Geometry Group은 ILSVRC-2014 대회에서 VGG-16 모델을 개발했습니다. 매우 정확하고 분류 및 탐지에 널리 사용됩니다. Caffe의 사전 훈련 된 모델에서이 버전을 수정했습니다. Darknet 고유의 이미지 전처리 (평균 빼기 대신 Darknet이 이미지를 -1과 1 사이에서 조정 함)에 맞추기 위해 추가로 6 개의 신기원을 훈련 받았습니다.

  • Top-1 Accuracy: 70.5%
  • Top-5 Accuracy: 90.0%
  • Forward Timing: 10.7 ms/img
  • CPU Forward Timing: 4.9 s/img
  • cfg file
  • weight file (528 MB)

Darknet19

  • Top-1 Accuracy: 72.9%

  • Forward Timing: 6.0 ms/img
  • CPU Forward Timing: 0.66 s/img

  • weight file (80 MB)
